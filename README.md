# Applied Machine Learning Portfolio

This folder organizes selected machine learning projects from MSE 5760 into a cleaner portfolio format for LinkedIn or GitHub.

## Project Focus

Applied machine learning for materials science, biomedical data analysis, and image-based classification.

## Included Reports

- `reports/Cancer_Type_Classification_TCGA_Report.pdf`  
  Cancer type classification from TCGA RNA-Seq gene expression data using PCA, stacked denoising autoencoders, and variational autoencoders.

- `reports/Ti64_Microstructure_Classification_Report.pdf`  
  Ti-6Al-4V microstructure image classification using CNN, MLP, PCA-based classical models, and HOG feature extraction.

- `reports/Materials_Bandgap_MLP_Regression_Report.pdf`  
  Materials bandgap prediction using multilayer perceptron regression models.

## Included Notebooks

- `notebooks/tcga_cancer_classification.ipynb`
- `notebooks/ti64_microstructure_classification.ipynb`
- `notebooks/materials_bandgap_mlp_regression.ipynb`

## Key Skills Demonstrated

- Python-based machine learning workflows
- Data preprocessing and feature engineering
- Classification and regression modeling
- CNNs for image classification
- MLPs for materials property prediction
- PCA, HOG, UMAP, autoencoders, and VAEs
- Model evaluation with accuracy, F1-score, RMSE, R2, confusion matrices, and error analysis
